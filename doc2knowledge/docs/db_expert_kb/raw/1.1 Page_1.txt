  场景二 
TPCH 场景， 因为TPCH 场景中的 SQL 语句， 基本都可以认为是慢SQL. 此处， 我们以该
类型的 SQL 语句为例， 进行测试， 看一下我们的功能是否可以覆盖TPCH 中的问题场景。 其
中，TPCH 的 22 条语句，其对应的慢 SQL 根因有很多论文在研究。因此，该根因也比较容
易对照正误。 
 
说明：上述场景只是为了对慢 SQL 根因诊断进行讲解，用户可以使用已有的场景进行测试。
