# 边界场景

## 场景描述

边界场景用于测试诊断引擎在特殊情况下的表现，包括误报检测、瞬时异常等。这些场景帮助验证系统的鲁棒性和准确性。

## 包含用例

| 用例ID | 用例名称 | 难度 | 描述 |
|--------|----------|------|------|
| case_19_idle | 空闲系统误报测试 | Easy | 验证系统不会对空闲状态产生误报 |

## 典型特征

- 指标处于正常范围
- 可能存在瞬时波动
- 需要区分真实异常和噪声

## 诊断要点

1. 确认指标是否真正异常
2. 分析异常持续时间
3. 排除误报可能性
4. 验证诊断结果的准确性
